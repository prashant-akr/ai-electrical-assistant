# Electrical Engineering AI Assistant

## Overview
This project is a simple AI-based assistant designed for electrical engineering students. It helps users with basic concepts, formulas, and viva preparation.

## Features
- Answers EE questions
- Provides formulas
- Supports viva questions
- Simple and fast interface

## How it Works
User enters a query → System processes keywords → Returns relevant response.

## Technologies Used
- HTML
- CSS
- JavaScript

## Assumptions
- Works on keyword-based logic
- Focused on basic EE concepts
